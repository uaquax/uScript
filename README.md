# uScript v0.0.1
uScript - is a scripts mamager.

## Commands
[x] - help
    Help menu. `--help`, `-h`

    $ ./uScript --help
    $ ./uScript -h
=

[x] - list
    List of scripts in the set directory. `--list` `-l`

    $ ./uScript --lsit
    $ ./uScript -l


[x] - path
    Set path to the scripts directory. `--path`, `-p`

    Example:
    $ ./uScript --path "/home/USER/scripts/"
    $ ./uScript -p "/home/USER/scripts/"

[x] - run
    Run a script. `--run`, `-r`

    Example:
    $ ./uScript --run HelloWorld
    $ ./uScript -r Example
